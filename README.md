## 1️⃣ Term Deposit Subscription Prediction (Bank Marketing)

### 🎯 Objective
Predict whether a bank customer will subscribe to a term deposit after a marketing campaign.

### 🔍 Key Tasks
- Data exploration and preprocessing
- Encoding categorical features
- Training classification models:
  - Logistic Regression
  - Random Forest
- Model evaluation using:
  - Confusion Matrix
  - F1-Score
  - ROC Curve
- Explainability using:
  - SHAP
  - LIME

### 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- SHAP
- LIME
- Matplotlib
- Seaborn

### 📈 Skills Demonstrated
